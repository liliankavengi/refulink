# Refulink Architecture

## Overview
Refulink is designed as a financial bridge integrating local mobile money (M-Pesa) with the Stellar network using Soroban smart contracts.

## Components
- **Backend (Django)**: Handles identity verification, M-Pesa webhooks, and interacts with the Stellar network.
- **Frontend (React Native)**: User interface for wallet management and transactions.
- **Blockchain (Stellar/Soroban)**: Smart contracts for secure value transfer and auditability.
