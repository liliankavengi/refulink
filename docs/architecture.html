🔐 Identity Loop

1
User enters Alien ID (RIN) in the Identity Vault screen.
2
Django SHA-256 hashes the RIN and checks it against the IPRS mock DB. Off-Chain
3
Backend issues a JWT token and creates a RefugeeIdentity with a generated Stellar keypair.
4
Soroban register_identity() stores hashed_rin + public key on the Stellar ledger. On-Chain
5
Ambassador QR scan triggers vouch() → set_verified(true); green badge appears in UI.

💸 M-Pesa Bridge (Mint)

1
User taps Deposit → sees Paybill number & account instructions.
2
User sends money via M-Pesa. Safaricom Daraja fires a C2B webhook. Off-Chain
3
Django matches phone to RefugeeIdentity, calls mint_tokens_for_deposit().
4
Stellar payment op sends KES token (ClickPesa issuer) to user's Stellar address. On-Chain
5
Wallet Dashboard balance updates via Horizon API on next refresh.

🔄 Withdrawal (Burn)

1
User taps Withdraw, enters phone number and amount.
2
burn_tokens_for_withdrawal() sends KES tokens back to the ClickPesa issuer. On-Chain
3
On success, Django triggers Daraja B2C payment to the user's phone number. Off-Chain
4
KES cash arrives in the user's M-Pesa wallet within seconds.

🔐 Security Layer

1
Stellar private key is generated server-side, returned ONCE to the client.
2
Client stores private key in expo-secure-store (hardware-backed keychain).
3
expo-local-authentication requires Face ID / Fingerprint before any key access.
4
All API calls require a Bearer JWT (60-min lifetime, silent refresh via interceptor).

📊 Trust & Credit Vault

1
Trust Score Engine reads on-chain vouches from VouchContract. On-Chain
2
Adds off-chain signals: tx volume, account age, verification status. Off-Chain
3
Score 0–100 → credit limit: 0 / 5K / 15K / 30K KES.
4
Borrow UI renders Gold/Silver/Bronze tier ring and loan slider up to the limit.

🌍 AI Language Layer

1
App launches with LanguageSelectionScreen (EN / SW / SO / AR).
2
Choice persisted to AsyncStorage via LanguageContext.
3
All UI strings rendered via t("key") hook with RTL support for Arabic.
4
Vouching agreements translated server-side via deep-translator (Google Translate).
