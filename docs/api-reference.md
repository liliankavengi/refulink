# API Reference

## Identity API
- `POST /api/identity/verify/`: RIN verification.
- `POST /api/identity/token/`: JWT auth.

## Wallet API
- `GET /api/wallet/balance/`: KES balance.
- `POST /api/wallet/transfer/`: Local transfer.

## Stellar Integration
- `POST /api/stellar/deposit/`: Deposit KES to Stellar.
- `POST /api/stellar/withdraw/`: Withdraw from Stellar to KES.
